# mvm_alterna_world
My Attempt At Recreating Splatoon Side Order As A MvM Mission

## How to install script

Copy all the `.nut` scripts into `[TF2 Folder]/tf/scripts/vscripts`. If the folders do not exist, create them.

## How to execute script

1. Load up a MvM map
1. Open up developer console
1. Run the following command `script_execute alterna_world_setup.nut`

## Commands

```
!chips - list all the chips available to the player.
!debugaddchip <internal_chip_name> - Increments the chip (`sv_cheats` must be set to 1)
!debugmaxoutallchips - Maxs out all the chips that are available to the player.
```
